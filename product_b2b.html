How to Use

Application Process:

  1. Preparation: Ensure the sizing machine is clean and properly calibrated
  2. Dilution: Mix the sizing agent with water according to the recommended ratio (1:10)
  3. Application: Apply the solution to warp yarns using the sizing machine
  4. Drying: Allow the sized yarns to dry completely before weaving
  5. Weaving: Proceed with the weaving process using the sized yarns

Recommended Dosage:

  • Light fabrics: 2-3% concentration
  • Medium fabrics: 3-5% concentration
  • Heavy fabrics: 5-8% concentration
