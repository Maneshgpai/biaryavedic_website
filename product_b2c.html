How to Use

Application Process:

  1. Preparation: Ensure your iron is clean and set to appropriate temperature
  2. Spray Application: Simply apply this spray starch to clothes while ironing
  3. Ironing: Experience the convenience of an anti-wrinkle spray for clothing
  4. Dark Colors: For dark colours, starch can be applied inside the clothes rather than visible
  5. Result: Enjoy smoother and more efficient ironing process

Usage Tips:

  • Shake well before use for best results
  • Hold spray bottle 6-8 inches from fabric
  • Apply evenly across the fabric surface
  • Allow to dry slightly before ironing
